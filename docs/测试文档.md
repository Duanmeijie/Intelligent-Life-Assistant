# 智能生活助手 - 测试文档

## 1. 测试目的

本测试文档旨在对**智能生活助手（Intelligent Life Assistant）** 项目进行全面系统的测试，确保系统各功能模块的正确性、稳定性和可用性。具体目标包括：

- 验证用户认证系统的完整流程（注册、登录、资料管理、密码修改）
- 验证日程管理模块的 CRUD 操作及状态流转功能
- 验证习惯追踪模块的创建、编辑、删除及状态管理功能
- 验证每日打卡模块的打卡、统计、撤销等核心功能
- 验证社交社区模块的动态发布、点赞、评论互动功能
- 验证 AI 智能推荐模块的分析与推荐准确性
- 验证数据可视化页面的图表展示正确性
- 验证系统的安全性、性能及异常处理能力

## 2. 测试范围

### 2.1 功能模块覆盖

| 模块 | 子功能 | 优先级 |
|------|--------|--------|
| 用户认证 | 注册、登录、获取/更新资料、修改密码 | P0-核心 |
| 日程管理 | 创建、编辑、删除、查询、状态更新 | P0-核心 |
| 习惯追踪 | 创建、编辑、删除、查询、活跃/暂停 | P0-核心 |
| 每日打卡 | 打卡、查询记录、统计查看、撤销打卡 | P0-核心 |
| 社交社区 | 发布动态、删除动态、动态列表、点赞、评论 | P1-重要 |
| AI 推荐 | 习惯推荐、效率分析、日程建议 | P1-重要 |
| 数据可视化 | 习惯图表、打卡热力图、日程分布 | P1-重要 |
| 仪表盘 | 关键指标展示、快捷操作 | P2-普通 |

### 2.2 测试类型

- **功能测试**：验证各功能模块的正确性
- **边界测试**：测试输入数据的边界值
- **异常测试**：测试错误输入和异常场景
- **兼容性测试**：验证不同浏览器和屏幕尺寸下的表现
- **安全性测试**：验证认证鉴权和数据隔离

   ```bash
   # 启动 MySQL 服务
   net start mysql

   mysql -u root -p < server/sql/init.sql
   
   # 启动 Redis 服务

   cd server
   npm install
   
   # 配置环境变量
   cp .env.example .env
   # 编辑 .env 文件填入实际的数据库连接信息
   ```
